又一款开源、跨平台、高颜值的 Markdown 笔记应用 —— Notable

2019-01-04

Notable 是 Github 上新发布的一款优秀的开源 Markdown 笔记应用，短短时间已经有 4770 多赞。Notable 有着简洁、优雅的三栏界面，使用可以无限层嵌套的标签（tags）管理 Markdown 文件。

Notable 的文档组织形式：

/path/to/your/data_directory
├─┬ attachments
│ ├── foo.ext
│ ├── bar.ext
│ └── …
└─┬ notes
  ├── foo.md
  ├── bar.md
  └── …

Notable 的特点：

  • 没有专有格式：只是一个结构如上所示的文件夹，笔记文件是纯 Markdown 文件，其元数据存储为 Markdown 文头注释。附件也是普通文件，如果您附加一个picture.jpg附件，它的所有内容都将被保留，并且它将像任何其他文件一样保持可访问状态。
  • 正确的编辑器：Notable 不使用任何 WYSIWYG 编辑器，你只需要写一些 Markdown，它就会被渲染为 GitHub 风格的 Markdown。内置编辑器是 CodeMirror，也可以在系统默认的 Markdown 编辑器中打开当前文件。
  • 无限可嵌套标签：在 Notable 中你可以有 root 标签（foo），无限可嵌套标签（foo/bar，foo/.../qux），当然，一个文件可以有多个标签。
  • 跨平台：目前支持Windows和macOS双平台。
